eBay Product Search Platform

Web Application Development Course Project

2020 Summer, USC

Designed and developed a nano-shopping eBay site which is a featureful, interactive web application including products search, conditional filter, details viewing, user wishlist and cart updates.

Deployed on GCP.

Technology used:

HTML CSS TypeScript xxx Angular Python Flask AJAX MySQL GCP

Features:

  • - User E-mail validation needed for sign up.
  • - Supporting user profile edit and photo upload.
  • - Follow system for following other users.
  • - Logged-in users are able to add comments to posts.
  • - Automatic comments refresh using AJAX.
